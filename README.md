# ФинТрек — Учёт расходов компании

Веб-приложение на Python (Flask) для учёта расходов компании.

## Запуск

1. Установите Flask:
```
pip install flask
```

2. Запустите приложение:
```
python app.py
```

3. Откройте в браузере: http://127.0.0.1:5000

## Возможности

- **dashboard** — общая статистика и диаграмма расходов по категориям
- **Добавить расход** — форма с полями: название, сумма, дата, категория, комментарий
- **Список расходов** — все записи с фильтром по категориям и удалением
- Данные сохраняются в файл `expenses.json`

## Структура проекта

```
expense-tracker/
```
